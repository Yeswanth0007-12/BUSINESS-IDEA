## 🎯 Example Usage

### Step 1: Add a Product
1. Click "Products" tab
2. Click "Add Product" button
3. Fill in details:
   - SKU: LAPTOP-001
   - Name: Gaming Laptop
   - Category: Electronics
   - Length: 40 cm, Width: 30 cm, Height: 5 cm
   - Weight: 3.5 kg
   - Monthly Volume: 100
4. Click "Save"

### Step 2: Add a Box
1. Click "Boxes" tab
2. Click "Add Box" button
3. Fill in details:
   - Name: Medium Box
   - Length: 45 cm, Width: 35 cm, Height: 10 cm
   - Cost: $2.50
4. Click "Save"

### Step 3: Run Optimization
1. Click "Optimize" tab
2. Click "Run Optimization" button
3. View results showing best box for each product

### Step 4: View Results
1. Click "History" tab to see past runs
2. Click "Leakage" tab to see space utilization
3. Click "Dashboard" to see overall metrics

## 🔧 Container Management

### View Running Containers
```bash
docker ps
```

### View Logs
```bash
# Backend logs
docker logs packoptima-backend

# Frontend logs
docker logs packoptima-frontend

# Database logs
docker logs packoptima-db
```

### Restart Services
```bash
docker compose restart
```

### Stop Services
```bash
docker compose down
```

### Start Services
```bash
docker compose up -d
```
